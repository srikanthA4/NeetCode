## 📘 What is DSA?

**Data Structures and Algorithms (DSA)** form the backbone of efficient programming:

* **Data Structures** → How data is stored (Array, Tree, Graph, etc.)
* **Algorithms** → How data is processed (Sorting, Searching, etc.)
